Some python packages we used in this project:

  • scrapy - to collect data from ITMO University site (in combination with beautifulsoup for parsing);
  • pyLDAvis for topic model visualization;
  • wordmesh - to build word cloud (with some hacks to support russian language)
  • pymorphy2 - to get normal forms of Russian words;
  • razdel - for tokenization
  • Dash and plotly, matplotlib, seaborn, Tableau - for plots;
  • gensim - we used this to produce author vectors, to build coauthors plot.
